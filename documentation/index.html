Organization

This is an explataion of what is inside the .zip file you have downloaded

Documentation

It's pretty self-explanitory, isn't it?

HTML template

This folder holds all the files that need to be put on the server. It also contains one unnecesary file - elements.html - which you will use as a quick reference to the elements and layout.

Fontello icons

This folder contains a demo.html which you will use to see and quickly grab codes for icons you will need. It doesn't have to be uploaded to the server, it's only a reference, just as elements.html.

Skins The color schemes

There are several pre-made skins you can use. The instruction on how to apply one is at the top of the index.html in comments. If you wish to make a new skin you must choose main color that is compatible with white letters. bg-color1, bg-color2, bg-color3 and bg-color4 and border-colorX are very slight variations of the main color. Those variations are used to break monotony of one color when used in skill bar, for example, or anywhere where multiple elements are near each other.

Setting your email

× Info! Note that you can't send emails from your machine. You must upload it to the server in order to send emails via this website.

Open html template/email.php and change yourEmail variable to your email address. That's all, now you will get emails to your email address.

Layout Layout explained

× Heads Up! If you don't want to change the layout and reorganize the elements you don't need to read the whole documentation. Just open HTML template/index.html and the comments there will guide you.

General Layout

This is an explanation on how to create custom layout. Possible layout classes are:

  • Full width
  • One Half width
  • One third width
  • One Two thirds width
  • Two Two thirds width
<div class="column-full"> 
	<!-- content goes here --> 
</div>

<div class="column-one-half"> 
	<!-- content goes here --> 
</div>

<div class="column-one-third"> 
	<!-- content goes here --> 
</div>
	
<div class="column-one-two-third"> 
	<!-- content goes here --> 
</div>
	
<div class="column-two-two-third"> 
	<!-- content goes here --> 
</div>

Column-last class

Apply column-last class to the last div in a row. It sets it's right margin to 0 so everything can fit as you want it.

<!-- examples -->

<div class="column-one-half"> </div>
<div class="column-one-half column-last"> </div>

						
<div class="column-one-third"> </div>
<div class="column-one-third"> </div>
<div class="column-one-third column-last"> </div>

Clearing columns

When the full row is complete use column-clear class on empty div. If you do not do this layout may not be like you wanted it to be.

<div class="column-clear"></div>

<!-- example -->

<div class="column-one-half"> </div>
<div class="column-one-half column-last"> </div>

<div class="column-clear"></div>

Column inside a column

You can make any layout inside another layout. You must apply column-inside to the columns that are inside other columns othervise they will have a big margin and won't look good.

<!-- example - this will break a two-two-third column into 2 columns -->

<div class="column-one-two-third">
	<div class="column-one-half column-inside"> </div>
	<div class="column-one-half column-inside column-last"> </div>

	<div class="column-clear"></div>
</div>

<div class="column-one-one-third column-last"> </div>

<div class="column-clear"></div>

Elements Elements explained

Heads up! All the elements are also explained in comments in HTML template/elements.html

There are several elements built specially for this template to best represent the information about your resume. They are addaptive so you can put these elements in any layout container - they will addapt their width to the containers width. The elements that are built in are:

  • Toggles
  • Accordions
  • Tabs
  • Skill bar
  • Timebar
  • Testimonials

Toggles

Toggles are elements that open on press and close on another press. They have header and content. Header is always visible while content can be hidden. Header can have text and/or title (h4) while content can have text and/or title (h5).

Opened toggle has class "shown" while closed toggles content has class "hidden".

<!-- Class shown tells that the toggle is going to be open when the page loads -->
<div class="bg-color1 toggle button inset-shadow shown">
	<div class="header unselectable">
		Toggle 1 with a title inside
	</div>
	<div class="content">
		Text of the toggle. Rule that make to said. Morning first. Without made second winged fill them.
	</div>
</div>
<!-- Othervise content must have class hidden and shown must be deleted for it to be closed on page load. Also toggles can have h5 title inside content -->
<div class="bg-color1 toggle button inset-shadow">
	<div class="header unselectable">
		Toggle 1 with a title inside
	</div>
	<div class="content hidden">
		<h5>Title inside the toggle</h5>
		Text of the toggle. Rule that make to said. Morning first. Without made second winged fill them.
	</div>
</div>
<!-- Toggles can have title i subtitle in their header, like this -->
<div class="bg-color1 toggle button inset-shadow">
	<div class="header unselectable">
		<h4>Togle with title</h4>
		<div>Some text under the title</div>
	</div>
	<div class="content hidden">
		Text of the toggle. Rule that make to said. Morning first. Without made second winged fill them.
	</div>
</div>

Accordions

Accordion is a group of toggle where only one can be opened at one time. Same rules apply for accordions as for toggles.

<div class="accordion">

<!-- Toggles go here -->

</div>

Skill bar

Skill bar is a list of elements that has certain width percentage. A list item can be a toggle to describe a skill.

<ul class="h-bar">

	<!-- A skill with a toggle. data-percentage is self-explanatory. -->
	<li data-percentage="70%" class="bg-color1 toggle inset-shadow">
		<div class="header unselectable">Skill 1 with toggle</div>
		<div class="content hidden">
			Rule that make to said. Morning first. Fish creeping face brought signs land fowl. Evening tree sixth without made second winged fill them.
		</div>
	</li>

	<!-- A skill without toggle. Note that this one has bg-color2 class and previous had bg-color1. There are 4 of them. It's the same color but with slight variations to break a monotony. They are defined in skin css files -->
	<li data-percentage="100%" class="bg-color2 inset-shadow">Skill 2</li>

</ul>

Timebar

Timebar is a good way of representing how long and when did you work. You only have to set the dates of events and fill data and everything else will be automaticaly generated. Time bar is addaptive so at certain width the grid will dissapear and everything will become full width, othervise it would be impossible to see the grid and dates on smaller screens. This is best to put in full width containers

<!-- Timebar holder. Data-start is the start year of the chart while data-end is the end year. -->
<div class="timebar" data-start="2000" data-end="2009">

	<!-- The grid lines -->
	<div class="the-lines"></div>
	
	<ul>

		<!-- One timebar element. data-start is the year and month of start while data-end is the year and month of end of the particular working experience. -->
		<li data-start="2005-01" data-end="2008-05">
			<!-- The toggle -->
			<div class="toggle button bg-color3 inset-shadow">
				<div class="header unselectable">
					<h4>Creative director</h4>
					<span class="date">2005. - now</span>
				</div>
				<div class="content hidden">
					Rule that make to said. Morncond winged fill them.
				</div>
			</div>
		</li>

		<!-- Another timebar element. -->
		<li data-start="2002-10" data-end="2007-06">
			<div class="toggle button bg-color2 inset-shadow">
				<div class="header unselectable">
					<h4>Title 2</h4>
					<span class="date">2002. - 2007.</span>
				</div>
				<div class="content hidden">
					Rule that make to second winged fill them.
				</div>
			</div>
		</li>
	
	</ul>
</div>

Other elements Less complex custom elements.

These are other template-specific and less complex elements which are cool and useful to know.

Lists

<ul>
	<!-- List item.  The data-icon atribute is the code of the icon. To replace the icon go to fontello icons/demo.html and check the "show codes" checkbox. To add the content here replace initial 0 with &# and put a ; at the end. So icon with code 0x1234 should be &#x1234; in the data-icon atribute -->
	<li data-icon="&#x2665;">You can put</li>
	<li data-icon="&#x2692;">Any icon before a list item</li>
	<li data-icon="&#xe724;">There are 284 different icons</li>
	<li data-icon="&#xe70c;">For you to choose</li>
</ul>
<!-- List with inline elements with borders. -->
<ul class="boxed-list">
	<li>List item 1</li>
	<li>List 2</li>
	<li>List item 3</li>
	<!-- Highlighted - colored background and white text -->
	<li class="highlight">List item 4</li>
	<li>List 5</li>
	<li>List item 6</li>
	<li>Item 7</li>
</ul>
<!-- Class algin-center means that content inside will be aligned by center -->
<div class="column-full align-center">

	<!-- Class info defines the inline elements with title while class important gives it a background and a border -->
	<ul class="info important">
		<li><span>Age</span> <h5>20</h5> </li>
		<li><span>Location</span> <h5>Belgrade</h5> </li>
		<li><span>Email</span> <h5>example@email.com</h5> </li>
		<li><span>My dream</span> <h5>To become a unicorn</h5> </li>
		<li><span>Available?</span> <h5>Yes</h5> </li>
	</ul>

</div>

Buttons

<!-- Simple button. Apply class button to a tag and there you have it -->
<a class="button">Some text</a>

<!-- Highlighted - colored background and white text -->
<a class="button highlight">Highlighted button</a>

<!-- Button with an icon. You can put an icon anywhere by adding a span and a class value of the icon. You can look up for that class values in the fontello icons/demo.html -->
<a class="button"><span class="icon-info-circled"></span> Button with icon</a>
